## Getting Started

### Clone the repo
```bash
git clone <your-repo-url>
cd bookstoreapi
```

---
### Frontend (Angular)

```bash
cd frontend
npm install
ng serve
# App runs at http://localhost:4200/
```

### Backend (Spring Boot)

```bash
cd src/main/java
# Run using your IDE or
mvn spring-boot:run
# API runs at http://localhost:8090/
```
